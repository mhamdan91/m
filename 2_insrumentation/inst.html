Some example systems at hand

Composter Control Box

Designed a traditional control box to control operation of 5HP AC motor. Safey measures are ensured by using an RSTP controller.
Technologies Used: Contactor, Mechanical protection, RSTP, Overloads.

Ground Sensing system

In this system we aim to evaluate the precision of various sensors(Lidar, radar, ultrasonic, laser) in distance measurement of ground through leafy material or canopy.
Technologies Used: NI-cRIO, Matlab, labVIEW, Analog sensors (Laser, ultrasonic), Digital sensors (radar, CAN ultrasonic, lidar).

Multifunctional Electronic System

This project is one of the most fun and interesting projects I have ever done while I was an undergradue. In this project I was only asked to implement a single device with a single functionality; however, due to my inquisitive mentality, I implemented a multifunctional systems that leverges the most of the device at hand (PCI microcontroller). The functionalities include: 1- a Piano with upto 8 notes, 2- Motoring: a security lock system to a bank vault where a password using Pin-pad used to open the door, 3- Multifunctional stop watch, 4- Alarm clock, 5- a touch system for multi-purpose functionality, 6- Lighting control sytem, 7- temperature control with a cooling system, 8- Pressure control system, 9- a handy calculator, 10- sound detection system.


Technologies Used: PIC18F, C , Flexi sensor, LDR, LEDS, Stepper motors, Fan, DSP
